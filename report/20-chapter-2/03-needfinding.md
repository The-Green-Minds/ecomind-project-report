## Needfinding

Los artefactos de esta sección no se completarán con imaginación ni con información demográfica genérica. Cada atributo deberá vincularse con las entrevistas registradas en la sección 2.2.

### User Personas

**Pendiente de entrevistas.** Se prevén, como máximo inicial, una Persona de díada familiar, una de facilitador y una de responsable de programa. Solo se separarán adulto y escolar en dos Personas si los datos muestran tareas, objetivos y patrones suficientemente distintos.

### User Task Matrix

**Pendiente de entrevistas.** La matriz describirá tareas existentes sin asumir EcoMind. Los verbos candidatos —planificar, elegir, ejecutar, evidenciar, revisar, interpretar y reportar— deberán confirmarse mediante observación o relato reciente.

| Tarea actual | Díada familiar | Facilitador | Responsable de programa | Evidencia de entrevista |
|---|---|---|---|---|
| Planificar o comprender una actividad | Pendiente | Pendiente | Pendiente | Pendiente |
| Ejecutar y acompañar | Pendiente | Pendiente | Pendiente | Pendiente |
| Registrar o revisar evidencia | Pendiente | Pendiente | Pendiente | Pendiente |
| Interpretar resultados | Pendiente | Pendiente | Pendiente | Pendiente |
| Preparar o usar un informe | Pendiente | Pendiente | Pendiente | Pendiente |

### User Journey Mapping

**Pendiente de entrevistas.** Se modelará primero el escenario As-Is y luego el To-Be. Cada fase incluirá `doing`, `thinking/saying`, `feeling`, pains y oportunidades.

### Empathy Mapping

**Pendiente de entrevistas.** Ninguna cita o emoción se atribuirá a una Persona sin registro trazable.

### Ubiquitous Language

El vocabulario del dominio se consolidará después de EventStorming y entrevistas. Los siguientes términos son candidatos técnicos de trabajo y no sustituyen el lenguaje utilizado por los participantes:

| Término candidato | Definición de trabajo | Estado |
|---|---|---|
| Sustainability Action | Ejecución de una actividad ambiental asociada a un reto o campaña. | Por contrastar con facilitadores. |
| Action Evidence | Registro que respalda que una acción ocurrió; no demuestra necesariamente impacto. | Por contrastar con los tres segmentos. |
| Environmental Observation | Valor ambiental con métrica, unidad, tiempo, fuente y calidad. | Por contrastar con especialistas y futuros datos IoT. |
| Impact Assessment | Resultado calculado con nivel de evidencia, método y versión. | Por validar; evitar lenguaje de impacto si no existe base. |
| Campaign | Conjunto temporal de objetivos, retos, participantes y criterios gestionado por una organización. | Por contrastar con responsables. |
| Facilitator | Persona que acompaña, asigna o revisa dentro de un programa. | Responsabilidades por delimitar. |
| Family Team | Grupo privado compuesto por escolar y adulto responsable. | Nombre por validar con familias. |
